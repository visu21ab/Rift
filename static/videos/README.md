# Landing Page Video

Place your landing page video file here.

## Supported Formats
- MP4 (recommended): `landing-video.mp4`
- WebM (optional): `landing-video.webm`

## Video Specifications
- Recommended resolution: 1920x1080 or 1280x720
- Recommended format: MP4 (H.264 codec)
- File size: Keep under 10MB for optimal loading
- Aspect ratio: 16:9 works best

## Usage
The video will automatically play on the landing page with:
- Autoplay (muted)
- Loop
- Responsive sizing

## Adding Your Video
1. Name your video file `landing-video.mp4`
2. Place it in this directory (`static/videos/`)
3. Optionally add a WebM version for better browser compatibility
